# Security Policy

## Supported Versions

Only the latest active version of the dataset and validation tools is supported with security updates and bug fixes.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please **do not** open a public issue. Instead, report it by:

1. Using the **Private Vulnerability Reporting** feature in the GitHub repository's Security tab (recommended).

We will acknowledge your report and coordinate a fix as quickly as possible.
